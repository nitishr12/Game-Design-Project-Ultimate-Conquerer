Controls:

Up - Jump
Left Arrow - Move Left
Right Arrow - Move Right
Ctrl - Kick
Space - Punch

Use the n key to move from one dialogue to the next during cutscenes.

Cheat:
Press Alt key to skip that level (there are three levels in total)
